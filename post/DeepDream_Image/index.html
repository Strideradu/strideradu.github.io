How to generate header image in my website using DeepDream, 用DeepDream提高网站逼格= =

2017-03-26

费了很大劲终于调好了站点，于是想着再学着一些网站加一个封面图，首先想到的是找个MSU的或者大济南的，可是找了半天没找到逼格够高的，这是突然瞅到了旁边的《Deep Learning》的封面。

看了下背面的注解，是一个叫Ddaniel Ambrosi的哥们用自己拍的全景照片再用谷歌的DeepDream神经网络处理的~逼格满满啊！！！于是冲到网站搞了他那副灯塔的逼格图~不过紧接着就发现人家图片上说版权所有不让乱用……嗯，那我们自己做一副！

Beaumont Tower是MSU的标志性建筑， 我用谷歌图片搜了一副比较满意的图片。然后下面进入DeepDream的在线生成器，开始创作！网站里可以选择Deep Style还是Deep Dream, style应该是模仿一些画家的风格。话不多说我进入了Deep Dream， 上传图片，有一些选项，因为我要高清图所以后面Depth只能选Normal。比较关键的是Neural Network Layer，决定了最后生成的风格。一开始我选的是inception_4a-pool，生成的图片好多眼睛似的玩意，看上去有点瘆得慌。我马上又换了conv2_3x3_reduce, 嗯这下看上去很像印象派风格了！搞定！

这幅题图既是MSU的标志建筑，还用上了最火的深度学习，产生了印象派风格的图片，达到了我期望的逼格！
